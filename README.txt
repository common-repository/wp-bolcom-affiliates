== Description ==

Bol.com Partnerprogramma by Biz2Web enables site owners to insert Bol.com Partnerprogramma links into any page or post.

== Installation ==

1. Upload the folder `wp-bolcom-affiliates` to the `/wp-content/plugins/` directory
2. Activate the plugin "Bol.com Partnerprogramma by Biz2Web" through the 'Plugins' menu in WordPress
3. Use option to insert link in post or page by clicking the Bol.com icon

== Frequently Asked Questions ==

= Where do I Find my Bol.com site ID? =

You can find your Bol.com site ID via:
https://partnerprogramma.bol.com/partner/affiliate/account.do

= I can only see a shortcode and nothing else happens. =

The plugin only works if a page or post is published.
Please make sure your content isn't in concept to see if it works.
If not, please contact us and let us know.
